## 📋 What's This?

A simple directory page that links to all my GitHub Pages projects in one place.

## 🛠️ Setup

This site deploys automatically via GitHub Pages. To enable:

1. Go to repository **Settings** → **Pages**
2. Under "Source", select **Deploy from a branch**
3. Choose `main` branch and `/ (root)` folder
4. Save
